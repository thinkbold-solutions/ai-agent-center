# AI Agent Platform for GoHighLevel

A multi-agent AI business operating system that integrates with GoHighLevel (GHL).

## Overview

This platform provides an AI-powered workforce including:

- **AI Receptionist** - Handle calls/texts 24/7
- **Finance Agent** - Business analytics chat
- **Personal Assistant** - Monitor your phone line
- **Social Media Manager** - Content & trends

## Documentation

See the `/docs` folder for detailed specifications.

## Tech Stack

- **Frontend:** React + Tailwind CSS
- **Backend:** Python FastAPI
- **Database:** MongoDB Atlas
- **Hosting:** Netlify
- **Mobile:** Capacitor (iOS/Android)
